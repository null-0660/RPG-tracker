## 🚀 Быстрый старт

### Вариант 1: Открыть в браузере

Просто откройте файл `index.html` в любом современном браузере.

### Вариант 2: Локальный сервер (рекомендуется для PWA)

**Через Python:**

```bash
python -m http.server 8080
# Откройте http://localhost:8080
```

**Через Node.js:**

```bash
npx serve .
# или
npx http-server -p 8080
```

**Через PHP:**

```bash
php -S localhost:8080
```

## 📱 PWA (Progressive Web App)

Приложение поддерживает:

* ✅ Работу оффлайн
* ✅ Установку на домашний экран телефона/планшета
* ✅ Быструю загрузку
* ✅ Push-уведомления

### Установка на телефон

**Android (Chrome):**

1. Откройте сайт в Chrome
2. Меню (3 точки) → "Установить приложение"

**iOS (Safari):**

1. Откройте сайт в Safari
2. Кнопка "Поделиться" → "На экран «Домой»"

## ✨ Функции

### 📇 Управление задачами

* Карточки задач с категориями (Учёба 📚, Спорт 💪, Творчество 🎨, Другое 📝)
* Система сложности (1–5 звёзд)
* Подзадачи для сложных проектов
* Фильтрация и сортировка

### 🌱 Привычки

* 6 готовых привычек для старта
* Приоритеты (высокий, средний, низкий)
* Серии выполнения с бонусами
* Ежедневный автоматический сброс

### ⏱️ Pomodoro Таймер

* 25 мин работа / 5 мин отдых
* Круговой индикатор прогресса
* Награда +10 XP за сессию
* Браузерные уведомления

### 🏅 Таблица лидеров

* Соревнование с другими игроками
* Подиум для топ-3 (🥇🥈🥉)
* 10 демо-игроков для старта
* Обновление в реальном времени

> **📌 Примечание:** сейчас таблица лидеров заполнена демо-данными для наглядности. Полноценная онлайн-версия с автоматическим обновлением рейтинга находится в разработке (в планах).

### ⚔️ Карточный бой

* 10 типов монстров: Гоблин, Орк, Скелет, Призрак, Дракон, Демон, Робот, Инопланетянин, Зомби, Вампир
* 3 уровня сложности
* Критические удары
* Награды: XP и Золото

### 🏪 Магазин

* Зелья: Лечение, Мана, Сила
* Бустеры: +50% XP, +50% Золота
* Скины: 6 вариантов

### 🏆 Достижения

* 23 штуки, с прогрессом и наградами

### 💬 Мотивация

* Цитата дня на дашборде
* 30+ цитат в 6 категориях

### 📊 Статистика

* Боевая статистика
* Эффективность по типам задач
* Недельная активность

## 🎮 Как играть

1. **Создание задачи** – "+" → заполняем форму → подзадачи (опционально)
2. **Выполнение задачи** – "✅ Выполнить" → получить XP и характеристики
3. **Карточный бой** – выбрать сложность и задачи → "⚔️ Атаковать" → получить награду
4. **Pomodoro** – выбрать задачу → "▶️ Старт" → работать 25 минут, отдых 5 минут

## 🔧 Технические детали

### Хранение данных

* Все данные через **localStorage**:

  * `liferpg_profile_v2` – профиль игрока
  * `liferpg_tasks_v2` – задачи
  * `liferpg_achievements_v2` – достижения
  * `liferpg_inventory_v2` – инвентарь
  * `liferpg_stats_v2` – статистика
  * `liferpg_habits_v2` – привычки
  * `liferpg_leaderboard_v2` – таблица лидеров

### Формулы

```js
level = Math.floor(totalXP / 100) + 1;
xpToNext = (level * 100) - (totalXP % 100);
statGain = Math.floor(taskXP / 20);
critChance = Math.min(30, luck * 1);
critDamage = damage * 1.5;
```

### Совместимость

* Chrome 80+, Firefox 75+, Safari 13+, Edge 80+
* Mobile browsers

## 🐛 Известные ограничения

1. Service Worker не работает через `file://` – используйте локальный сервер
2. localStorage ограничен ~5–10 MB
3. Синхронизация между устройствами отсутствует – используйте экспорт/импорт
