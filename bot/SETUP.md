# 🚀 Инструкция по запуску Telegram бота

## Шаг 1: Получение токена бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. BotFather попросит ввести имя бота (например: `LifeRPG Bot`)
4. Затем попросит ввести username (например: `LifeRPG_bot`)
5. После создания вы получите **токен** вида:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```
6. **Скопируйте этот токен!**

## Шаг 2: Установка Python

Если у вас нет Python:

1. Скачайте с [python.org](https://www.python.org/downloads/)
2. Установите версию **3.8 или выше**
3. При установке отметьте галочку **"Add Python to PATH"**

## Шаг 3: Настройка проекта

1. Откройте папку `bot`
2. Откройте файл `config.py` в текстовом редакторе
3. Замените `YOUR_BOT_TOKEN_HERE` на ваш токен:
   ```python
   BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
   ```
4. Сохраните файл

## Шаг 4: Установка зависимостей

Откройте терминал (командную строку) в папке `bot`:

```bash
cd bot
pip install -r requirements.txt
```

Или вручную:
```bash
pip install aiogram aiofiles aiosqlite python-dotenv
```

## Шаг 5: Запуск бота

В терминале в папке `bot`:

```bash
python bot.py
```

Вы должны увидеть:
```
2024-03-18 12:00:00 - root - INFO - База данных подключена
2024-03-18 12:00:00 - root - INFO - Бот запущен...
```

## Шаг 6: Проверка работы

1. Откройте Telegram
2. Найдите своего бота по username
3. Нажмите **Start** или отправьте `/start`
4. Бот должен ответить приветствием!

## 🐛 Решение проблем

### Ошибка: "No module named 'aiogram'"
```bash
pip install aiogram
```

### Ошибка: "Token is invalid"
- Проверьте токен в `config.py`
- Убедитесь, что нет лишних пробелов
- Токен должен быть в кавычках

### Бот не отвечает
1. Проверьте интернет соединение
2. Убедитесь, что токен правильный
3. Проверьте логи в файле `bot.log`
4. Перезапустите бота (Ctrl+C, затем снова `python bot.py`)

### Ошибка доступа к базе данных
- Убедитесь, что папка `data` существует
- Проверьте права доступа к папке

## 📱 Команды для проверки

После запуска отправьте боту:

```
/start - Приветствие
/help - Справка
/profile - Профиль
/addtask - Добавить задачу
/addnote - Добавить заметку
```

## 🔄 Остановка бота

Нажмите **Ctrl + C** в терминале для остановки

## 📝 Логи

Все события записываются в `bot.log`

Для просмотра в реальном времени:
```bash
# Linux/Mac
tail -f bot.log

# Windows
Get-Content bot.log -Wait -Tail 50
```

## 💡 Советы

1. **Хостинг**: Для постоянной работы разместите бота на сервере
   - Heroku, Railway, VPS
2. **База данных**: Регулярно делайте бэкап `data/database.db`
3. **Токен**: Никогда не публикуйте токен в открытом доступе!
