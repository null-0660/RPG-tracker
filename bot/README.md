# 🤖 LifeRPG Telegram Bot

Полнофункциональный Telegram бот с системой задач, RPG элементами и заметками.

## 📋 Возможности бота

### 🎮 LifeRPG функционал:
- ✅ Создание и выполнение задач
- ✅ Система уровней и XP
- ✅ Характеристики (Интеллект, Сила, Креатив, Удача)
- ✅ Инвентарь и магазин
- ✅ Достижения
- ✅ Статистика игрока

### 📝 Заметки (как Obsidian):
- ✅ Текстовые заметки
- ✅ Фото заметки
- ✅ Голосовые заметки
- ✅ Видео заметки
- ✅ Файлы
- ✅ Категории заметок
- ✅ Поиск по заметкам
- ✅ Теги

## 🚀 Быстрый старт

### 1. Получение токена бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Введите имя бота (например: `LifeRPG Bot`)
4. Введите username бота (например: `LifeRPG_bot`)
5. Скопируйте полученный токен

### 2. Установка зависимостей

```bash
cd bot
pip install -r requirements.txt
```

### 3. Настройка

Откройте `config.py` и вставьте ваш токен:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

### 4. Запуск

```bash
python bot.py
```

## 📱 Команды бота

### Основные:
```
/start - Запуск бота и регистрация
/help - Помощь и список команд
/menu - Главное меню
```

### Задачи:
```
/tasks - Список задач
/addtask - Добавить задачу
/complete - Выполнить задачу
/delete - Удалить задачу
```

### Профиль:
```
/profile - Мой профиль
/stats - Статистика
/achievements - Достижения
```

### Заметки:
```
/notes - Список заметок
/addnote - Добавить заметку
/photo - Фото заметка
/voice - Голосовая заметка
/file - Файл заметка
/search - Поиск заметок
/deletenote - Удалить заметку
```

### Магазин:
```
/shop - Магазин
/buy - Купить предмет
/inventory - Инвентарь
```

## 🔧 Технологии

- **Python 3.8+**
- **aiogram 3.x** - асинхронный фреймворк для Telegram Bot API
- **SQLite** - база данных
- **AIOFiles** - асинхронная работа с файлами

## 💡 Особенности

### Заметки как в Obsidian:
- **Связи между заметками** через `[[название]]`
- **Теги** через `#тег`
- **Категории** для организации
- **Поиск** по содержимому
- **Экспорт** в Markdown

### RPG элементы:
- Задачи дают **XP** и **золото**
- Характеристики растут от выполнения задач
- **Магазин** с бустерами и предметами
- **Достижения** за прогресс

## 🔐 Безопасность

- Данные хранятся локально в SQLite
- Поддержка нескольких пользователей
- Резервное копирование базы данных

## 📊 База данных

Бот использует SQLite с таблицами:
- `users` - пользователи
- `tasks` - задачи
- `notes` - заметки
- `achievements` - достижения
- `inventory` - инвентарь

## 🐛 Решение проблем

### Бот не запускается:
1. Проверьте токен в `config.py`
2. Убедитесь, что установлен Python 3.8+
3. Проверьте установку зависимостей: `pip install -r requirements.txt`

### Бот не отвечает:
1. Проверьте интернет соединение
2. Убедитесь, что токен актуален
3. Перезапустите бота
