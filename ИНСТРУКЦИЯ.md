## 📦 Варианты запуска

### 1️⃣ Локально на компьютере

#### Через Python:
```bash
cd project
python -m http.server 8080
```
Откройте: http://localhost:8080

#### Через Node.js:
```bash
cd project
npx serve .
```

#### Через PHP:
```bash
cd project
php -S localhost:8080
```

#### Просто открыть файл:
Дважды кликните на `index.html` (некоторые функции могут не работать).

---

### 2️⃣ GitHub Pages (бесплатный хостинг)

#### Шаг 1: Создайте репозиторий
1. Зайдите на https://github.com
2. Нажмите "New repository"
3. Название: `liferpg`
4. Тип: Public
5. Нажмите "Create repository"

#### Шаг 2: Загрузите файлы
1. Нажмите "uploading an existing file"
2. Перетащите **все файлы из папки `project`**
3. Нажмите "Commit changes"

#### Шаг 3: Включите Pages
1. Settings → Pages
2. Source: main branch
3. Нажмите Save
4. Через 1-2 минуты получите ссылку:
   ```
   https://ваш-username.github.io/liferpg/
   ```

---

### 3️⃣ Netlify (ещё проще!)

1. Зайдите на https://netlify.com
2. Зарегистрируйтесь (бесплатно)
3. Перетащите папку `project` в окно браузера
4. Через 30 секунд получите ссылку вида:
   ```
   https://liferpg-xxxx.netlify.app
   ```

---

### 4️⃣ Vercel

1. Зайдите на https://vercel.com
2. Зарегистрируйтесь через GitHub
3. Импортируйте репозиторий

## 💾 Сохранение данных

### Где хранятся данные?
- В **localStorage** вашего браузера
- ~5-10 MB места
- Автоматически сохраняются

### ⚠️ Важно!
- Данные привязаны к **браузеру и устройству**
- При очистке кэша данные **сотрутся**
- Для переноса используйте **Экспорт/Импорт** (в Профиле)

### Экспорт данных:
1. Откройте раздел "Профиль"
2. Нажмите "Экспорт данных"
3. Сохраните JSON файл

### Импорт данных:
1. Откройте раздел "Профиль"
2. Нажмите "Импорт данных"
3. Выберите сохранённый JSON файл

## 🐛 Если что-то не работает

### 1. Обновите страницу
```
Ctrl + F5 (Windows)
Cmd + Shift + R (Mac)
```

### 2. Очистите localStorage
Откройте консоль (F12) и введите:
```javascript
localStorage.clear();
location.reload();
```
⚠️ Данные сотрутся!

### 3. Проверьте консоль
Нажмите F12 и посмотрите, есть ли ошибки.

### 4. Попробуйте другой браузер
Chrome, Firefox, Edge, Safari

## 🎯 Советы по использованию

### Для новичков:
1. Начните с маленьких задач (сложность 1-2)
2. Выполняйте 3 задачи в день (ежедневный квест)
3. Поддерживайте серию дней (стрик)

### Для продвинутых:
1. Используйте Pomodoro для фокусировки
2. Участвуйте в боях за золото
3. Собирайте все достижения
4. Следите за статистикой
